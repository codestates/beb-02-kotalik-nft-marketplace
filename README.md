## Get Started

### Prerequisites

> **_설치된 modules_**

```
npm i semantic-ui-react semantic-ui-css
```

```
npm i caver-js
```

```
npm i web3
```

```
npm i ipfs-http-client
```

```
npm i assert
```

### Installation

```
cd client/
```

```
npm install
```

### Run Start

---

```
cd client/
```

```
npm run dev
```
